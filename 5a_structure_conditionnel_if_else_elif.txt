# La structure conditionnel 
if "si" 

Le code est executé seulement si la condition vérifier est vrai (True)
Alors le code s'éxecute. 
Si la condition vérifier est faux (False) le code n'est pas éxectuer.

if condition ..
    code ..


age = 20 
if age >= 18 :                  >> True donc le code print suivant ce lance car 20 est plus grand ou égal à 18
    print("Vous êtes majeur !")

age = 15 
if age >= 18 :                  >> False donc le code print ne se lance pas car n'est pas plus grand ou égal à 18
    print("Vous êtes majeur !")

elif "autrement si" 

Si mon premier if est False alors on passe a la condition suivante

#Ici ma condition if est fausse (False) donc le premier print n'est pas lancer, il va donc verifier la deuxième condition
#elif étant une condition vrai (True) il execute le deuxième code print("vous êtes mineur")
age = 15
if age >= 18 :
    print("Vous êtes majeur !")
elif age < 18 :
    print("Vous êtes mineur")

>> 'Vous êtes mineur'

#Dans le cas ou la premiere condition if est vrai (True), la deuxième condition n'est plus vérifier.
age = 20
if age >= 18 :
    print("Vous êtes majeur !")
elif age < 18 :
    print("Vous êtes mineur")

>> 'Vous êtes majeur'
